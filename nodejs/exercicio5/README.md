## Veículos API REST

Para o funcionamento da API basta instalar as dependências neste mesmo diretório:
```npm install```

Ápos instalação basta iniciar o server:
```npm start```

## Testes
Para a execução de testes foi criado um script:
```npm tests```

ou acessar o diretório de testes:
```cd tests```

e executar:
```mocha```
